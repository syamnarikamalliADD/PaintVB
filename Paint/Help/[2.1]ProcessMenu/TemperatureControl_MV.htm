Process Menu

Temperature Control

This screen is used to configure and monitor the material temperature control.

Note: Changes to this screen can only be made by a user with sufficient access level.

Click the area where you want help.

Description of Screen Controls

The following table provides a functional description of the various screen objects:

ITEM DESCRIPTION
Close Click on this button to close this screen and return to the main menu.
Save Click on this button to save changes to the PLC and database.
Undo Click on this button to cancel current edits. This does NOT apply to changes that have already been saved.
Print Click on this button to print the data displayed.
Change Log Click on this button to display the Change Log screen. Clicking the arrow to the right of this button displays a menu from which to choose 'Last 24 Hours', 'Last 7 Days', and 'All Available'. Selecting one of these options will display data for the associated length of time.

Click here for more information on the Change Summary screen.

Multi-View This button allows the user to toggle between Single-View and Multi-View modes. In Multi-View mode, the user may select multiple robots to view data for simultaneously. Alternatively, Single-View mode displays data for only one robot at a time.
Add When in multi-vew mode, this button adds the selected robot, controller, or tracking schedule.  A drop-down menu allows you to add groups of robots, controllers, or schedules. 
Clear When in multi-view mode, this button clears all the displayed columns.
Status The Status button displays/hides a window where information regarding data read and write operations is logged.
AutoRefresh This button allows the user to enable and disable the screen auto update feature. When enabled, the button is displayed in the pushed in state, and automatically updates the data displayed automatically. When disabled, the button is shown in a normal state. Disabling this feature allows the user to freeze the data displayed so that it may be recorded or printed.
Utilities The Utilities button opens a dropdown menu that offers import from or export to a spreadsheet.
Zone This dropdown box allows the user to select the appropriate zone. If only one zone exists, then only one zone will be available.
Controller This is a dropdown box from which the desired robot is selected. In Single-View mode, the display is updated whenever a robot is selected. In Multi-View mode, this selection is used by the Add button.
Temperature Setup Items

Each tab displays the robot setup items applicable to that area. The user may either enter or select a new value.

Note: Changes will appear in red, and will not take effect until the Save button is pressed.
TCU Enable Enables Temperature Conditioning.
Enables Temperature Conditioning. TCU Set-point value. Valid range for the Temperature Set-point is anywhere between the "Meter Low Temp Warning Threshold" and the "Meter High Temp Warning Threshold". If values outside of this range are used, the software will automatically reduce the set-point to the appropriate limit.

Current Meter Temp

 Current meter temperature in degrees Fahrenheit.
Meter Temp High/Low Alarm Meter temperature fault threshold in degrees Fahrenheit.
Meter Temp High/Low Warning Meter temperature warning threshold in degrees Fahrenheit.
Gun Temp High/Low Alarm Gun temperature fault threshold in degrees Fahrenheit.
Gun Temp High/Low Warning Gun temperature warning threshold in degrees Fahrenheit.
Current Gun Temp Current Gun Temperature in degrees Fahrenheit.
Viscosity Control Enables Viscosity Compensation. True = Enabled, False = Disabled. The TCU must also be enabled for the viscosity compensation to work.
Viscosity Setpoint Calibrated for nominal ~ 100% set-point.
Current Viscosity Material viscosity is typically calibrated so that the material has optimal application properties when the Viscosity Set-point is 100%. The current viscosity is expressed as a percentage. If the current viscosity is 86% and the Viscosity Set-point is 100%, the material will be chilled slightly to increase the viscosity until the set-point is reached. The Current Viscosity is only calculated during material circulation.
This indicator, located at the bottom of most screens that communicate directly with robots, provides information regarding the status of communications with each robot in the system. The indicator is green when communicating properly, red when not communicating, and yellow when a connection is being established. When the mouse pointer is held over one of these icons, a tooltip is displayed that indicates which robot it applies to.

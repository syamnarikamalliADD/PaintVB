E-stat Presets

The purpose of this screen is to view/edit the E-Stat Presets for each Style/Colour combination in each robot.

Presets are spcified in Kilovolts (KV).

All E-Stat presets are read from and written to the robot controller.

Note: Changes to this screen can only be made by a user with sufficient access level.

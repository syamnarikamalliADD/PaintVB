Diagnostics Manager

This screen allows the user to manage the diagnostics feature on the robot.

  • System - Enable recordings, select the current schedule and determine how long to keep diagnostic files on the PAINTworks computer.
  • Schedules - Setup or view the diagnostics schedule configurations on each robot.
  • Items - Setup or view the diagnostics item configurations on each robot.

System Configuration

  • Days To Keep/DMON Files - This controls how long DMON files are saved on the PAINTworks PC.
  • Days To Keep/Archive Zips - Each day's DMON files are saved to a zip file. This setting determines how long the zip archives are kept on the PAINTworks PC.
  • Enable - Enables automatic recording of diagnostics by the robot controller and copying to the PAINTworks PC.
  • Job or CC - Selects recording during the job execution or during color change cycles.
  • Schedule - Selects the schedule to record.

Note: Changes to this screen can only be made by a user with sufficient access level.

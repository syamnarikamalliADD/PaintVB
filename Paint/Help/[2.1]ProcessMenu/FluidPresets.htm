Fluid Presets

The purpose of this screen is to view/edit the Fluid Presets for each color in each robot. Fluid Presets consist of paint flow (cc/Min), turbine speed (krpm) or atom air (psi), and shaping air (slpm) or fan air(psi). All Fluid Presets are read from and written to the robot controller.

Note: Changes to this screen can only be made by a user with sufficient access level.

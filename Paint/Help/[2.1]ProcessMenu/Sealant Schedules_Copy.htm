Process Menu

SealantcSchedules Copy

This screen allows the user to copy sealant schedules to one or more robots. The yellow status box will assist the user in making all necessary selections. When appropriate selections have been made, 'Press Copy Button to begin copy of selected schedules' will appear in the box. The user may then press the 'Copy' button to perform the operation.

Note: Changes to this screen can only be made by a user with sufficient access level.

Click the area where you want help.

Refer to the Description of Screen Controls section below for more information.

Description of Screen Controls

The following table provides a functional description of the various screen objects:

ITEM

DESCRIPTION

Close

Click on this button to close this screen and return to the main menu.

Copy

Click on this button to copy changes to the selected robots after the desired selections are made.

Status

Click on this button to show or hide the status box.

From Robot

Use these checkboxes to select the robot FROM which sealant schedules will be copied.

From Material

If the robots have multiple dispensers each, this box contains a checkbox for each associated material. Use these checkboxes to select the material FROM which sealant schedules will be copied. If the robots have only one dispenser, 'Material 1' is automatically selected.

From Schedule

 Use these checkboxes to select the schedule number(s) to copy. If multiple schedules are selected, then the same schedules will be selected in the 'To Schedule' box.

To Rtobo

The Status button displays/hides a window where information regarding data read and write operations is logged.

To Material

This dropdown box allows the user to select the appropriate zone.  If only one zone exists, then only one zone will be available.

To Schedule

This is a dropdown box from which the desired robot is selected.

Schedule

Click on the Up/Down arrows to increment/decrement the schedule number as desired.  A schedule number may be entered manually by clicking on the black box that displays the current schedule number.
   

This indicator, located at the bottom of most screens that communicate directly with robots, provides information regarding the status of communications with each robot in the system.  The indicator is green when communicating properly, red when not communicating, and yellow when a connection is being established.  When the mouse pointer is held over one of these icons, a tooltip is displayed that indicates which robot it applies to.

Click Close to return to the Sealant Schedules screen.

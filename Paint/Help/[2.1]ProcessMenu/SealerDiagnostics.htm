Sealer Diagnostics

This screen allows the user to run system diagnostics utilities. The ‘Test Summary’ page (shown below) indicates the last time that each diagnostics test was successfully run, what user was logged in when it was run, when it is scheduled to be run next, and an ‘Ok’ or ‘Overdue’ status for each. To display the window for a particular test, click on the desired row (the row will be highlighted when the mouse if over it).

Refer to the Description of Screen Controls section below for more information.

Process Menu

Color Change

This screen is used to setup the color change cycles for each robot and each color valve.

The color change steps are entered/edited from a graphical representation or a grid display. The operator can set presets for each color change step from the Presets button or grid display. Each preset has fluid flow, turbine speed or atom air, and shaping air or fan air values. The presets are stored by robot, not by valve and can be copied from one robot to another robot.

The Step:Action/Event list and the grid display support context menus opened by right clicking on a step.
The context menus allow the user to inert or delete steps and change the action and/or event associated with an existing step.
The maximum number of color change cycle steps available for any color change cycle in the PAINTworks system is 100.

All Color Change Cycles are read from and written to the robot controller.

Note: Changes to this screen can only be made by a user with sufficient access level.

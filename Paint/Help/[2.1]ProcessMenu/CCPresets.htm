Color Change Presets

The purpose of this screen is to view/edit the color change presets for each robot. Color Change Presets consist of paint flow (cc/Min), turbine speed (krpm), and shaping air (slpm). All Fluid Presets are read from and written to the robot controller.

Note: Changes to this screen can only be made by a user with sufficient access level.

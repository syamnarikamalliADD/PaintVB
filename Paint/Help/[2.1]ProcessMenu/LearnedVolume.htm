Process Menu

Learned Volume

This screen allows the user to view/edit the learned volumes for each configured style. The 'Learned' volume is the amount of sealant expected to be used for a particular style.

Note: Changes to this screen can only be made by a user with sufficient access level.

Click the area where you want help.

Refer to the Description of Screen Controls section below for more information.

Description of Screen Controls

The following table provides a functional description of the various screen objects:

ITEM

DESCRIPTION

Close

Click on this button to close this screen and return to the main menu.

Save

Click on this button to save changes for the selected robot.

Undo

Click on this button to cancel current edits.  This does NOT apply to changes that have already been saved.

Print

Click on this button to print the displayed volumes.

Change Summary

Click on this button to display the Change Log screen. Clicking the arrow to the right of this button displays a menu from which to choose 'Last 24 Hours', 'Last 7 Days', and 'All Available'. Selecting one of these options will display data for the associated length of time.

Click here for more information on the Change Summary screen.

Status

The Status button displays/hides a window where information regarding data read and write operations is logged.

Zone

This dropdown box allows the user to select the appropriate zone.  If only one zone exists, then only one zone will be available.

Device

This is a dropdown box from which the desired robot is selected.

Style

This is a dropdown box from which the desired style is selected.

Learned Volume Alarms

This area provides a checkbox from which to enable and disable volume faults and a textbox to enter the percentage above and below which an alarm is posted. When volume faults are enabled, if the actual volume dispensed is greater or less than the learned volume by the specified percentage, the system faults and an alarm is posted. If disabled, only the alarm is posted.

Learned Volumes

This area displays the saved learned volumes.

Area Volumes

This area displays the saved area volumes.

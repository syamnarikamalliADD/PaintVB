Configuration Menu

Job Setup - Style ID

The purpose of this screen is to configure job setup information. Tabs are used to select each job setup parameter: Style/Option, Style ID

Styles

Each style has a photo-eye configuration associated with it.

The Style ID information is saved to the PLC. The PLC uses this data to verify that the job at the ID stations matches the data sent by the plant.

Note: Changes to this screen can only be made by a user with sufficient access level.

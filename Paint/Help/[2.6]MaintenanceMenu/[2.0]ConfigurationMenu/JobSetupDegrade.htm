Degrade

The purpose of this screen is to assign a degrade number that indicates which robot is bypassed when initiating the motion controllers. The last two digits of the job number will indicate which robot is disabled for the degrade option. With the proper access level, a user can restore the degrade option data from either the database, the PLC.

The FANUC degrade option is saved to the PLC. The PLC uses the robot's required data to initiate the motion controllers with the correct degrade job number . 

Note: Changes to this screen can only be made by a user with sufficient access level.

Maintenance Menu

Network Configuration

This screen is for Ethernet network diagnostics. It provides access to the local host file and common network commands.

Network Commands

  • Fanuc Ping - Uses the Fanuc robot server to verify the robot communication.
  • Ping - Verify the network can reach the device.
  • Arp - Lists IP addresses and associated MAC addresses.
  • Traceroute - Shows the route the network takes to a device.
  • IP Config - Displays the configuration of network adapters on the computer.

BOOTP and TFTP servers for robot image backups

The following functions manage the BOOTP and TFTP servers running on the computer to support the image backup and restore feature on the robot controllers. The servers are normally running on the PAINTworks computer. These buttons can be used to start or stop the servers.

  • Update BOOTP - Reads the configuration files and updates the network settings for the servers.
  • TFTP/BOOTP Status - Display the status of the servers (running/not running).
  • Start TFTP/BOOTP - Start the servers.
  • Stop TFTP/BOOTP - Stop the servers.

Host Names File - BOOTP Setup - TFTP Setup

The lower window allows you to view or edit network configuration files.

  • Host Names File - This file has the network addresses for all devices in the system.
  • BOOTP Setup - This file configures the BOOTP server to associate IP addresses with the MAC addresses of the robot controllers. When a robot controller CPU is changed this file needs to be updated. If PAINTworks is communicating with the robot normally, you can press the "Auto Configure" button to set up this file. You can also manually enter the MAC address. Save the file then use the "Update BOOTP" button to apply the changes.
  • TFTP Setup - This file configures the TFTP server. The "Auto Configure" button can be used to fill in the computer name and IP address. Save the file then use the "Update BOOTP" button to apply the changes.

Note: Changes to this screen can only be made by a user with sufficient access level.

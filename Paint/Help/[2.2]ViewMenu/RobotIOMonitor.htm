Robot I/O Monitor

This screen is used to monitor robot I/O, simulate robot I/O, and modify I/O comments.

The left side lists a group of I/O points by type. The auto-refresh button applies to items on the left. The I/O monitor feature on the right runs a faster update for the selected I/O points.

Note: Changes to this screen can only be made by a user with sufficient access level.

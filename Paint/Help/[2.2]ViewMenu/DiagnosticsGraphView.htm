Diagnostics Graph View

This screen allows the user to view detailed graphical data of a job.
The chart tab allows control of the time scale(horizontal) and selection of which items to chart.
The configuration tab allows the user to adjust the scaling of each item(vertical).
The other tabs show data saved in the DMON log.

Note: Changes to this screen can only be made by a user with sufficient access level.

View Menu

Bulk Supply Monitor

This screen displays the status of the bulk material supply system. The user may click on the desired tab to view the status of the associated drum.

Note: Changes to this screen can only be made by a user with sufficient access level.

Refer to the Description of Screen Controls section below for more information.

When the ‘Remote/Local’ selector switch for a drum is in the ‘Remote’ position, the ‘Prime Pump’, ‘Pump Reset’, and ‘Pump On’ graphics operate as pushbuttons that may be clicked on to perform the same function as the local pushbuttons. The ‘Depressurize’ mushroom pushbutton graphic may be clicked on anytime to depressurize the system.

When the ‘Drum Alternate’ feature is enabled, each drum is offline for the number of minutes set in the ‘Drum On Time’ textbox. This allows some material to be used from each drum repeatedly. This can be useful when the material supply header is in line with the drums, causing more material to be drawn from the first drum than the others.Caution must be observed when utilizing this feature, however, to avoid the situation where all drums are empty at the same time.

The user may also enable/disable the Auto Depressurize function using the ‘Auto Depressurize Enabled’ checkbox.When enabled, the bulk supply system will automatically depressurize when the client robots have been idle for the preset time period.If multiple robot cells utilize the same supply system, this feature must be enabled on each, and the bulk supply will not depressurize until all client cells have been idle for their preset times.

The ‘Pump Count Status’ area for each drum displays the number of pump cycles since the last reset. These counts should be reset, by clicking on the ‘Pump x Count Reset’ pushbuttons, after maintenance has been performed.When the ‘Actual’ count reaches the ‘Notify’ count, a popup reminder is displayed indicating that it is time to perform the required maintenance.

Description of Screen Controls

The following table provides a functional description of the various screen objects

ITEM

DESCRIPTION

Close

Click on this button to close this screen and return to the main menu.

Save

Click on this button to save changes to the PLC and database.

Undo

Click on this button to cancel current edits. This does NOT apply to changes that have already been saved.

Change Summary

Click on this button to display the Change Summary screen. Clicking the arrow to the right of this button displays a menu from which to choose 'Last 24 Hours', 'Last 7 Days', and 'All Available'. Selecting one of these options will display data for the associated length of time.

Status

The Status button displays/hides a window where information regarding data read and write operations is logged.

Zone

This dropdown box allows the user to select the appropriate zone. If only one zone exists, then only one zone will be available.

Drum 'x' Status

Displays the status of the associated drum.

Auto Depressurization

This area displays the auto depressurize settings.

Auto Depressurize Enabled

When this checkbox is checked, the bulk supply system will automatically depressurize after the preset number of hours and minutes of system idle time.

Preset

This is where the user enters the desired idle time in hours and minutes.

Actual

This is the current idle time. When the number of hours and minutes equal the preset number of hours and minutes, the bulk supply system will automatically depressurize.

Bulk Supply Depressurization

Displays the status of the mushroom type depressurize pushbutton on the local control panel for the associated drum. This graphic may also be clicked on to depressurize the system.

Note:

Clicking on this graphic (or pressing the actual button on the control panel) depressurizes the entire supply system, not just the associated drum.

System Pressurized

This graphic displays the status of the associated light on the supply drum's local control panel. This light is on when the system is pressurized and the drum is online.

Pump Active

This graphic displays the status of the associated light on the supply drum's local control panel. This light is on when the system is pressurized and the drum is online and flashes when the drum is currently being primed.

Air Press On

This graphic displays the status of the associated light on the supply drum's local control panel. This light is on when there is sufficient air pressure to the system.

Drum Present

This graphic displays the status of the associated light on the supply drum's local control panel. This light is on when a drum is in place.

Drum Ready

This graphic displays the status of the associated light on the supply drum's local control panel. This light is on when the drum's ram (plunger) is inserted into the drum.

Prime Pump

Click on this button to initiate a drum prime operation. This must only be performed upon initial load of a new drum. The prime operation will end automatically when the pumps stroke 10 times. If there is not enough room to pump 10 times, then the Pump Reset button may be pressed to end the prime.

Pump Reset

Click on this button to manually end a drum prime operation, or to take a drum offline.

Pump On

Click on this button to put a drum online.

Local / Remote SS

This graphic displays the status of the Local / Remote selector switch on the drum's local control panel. The Depressurization, Prime, Reset, and Pump On pushbuttons on this screen will only function when this switch is in the Remote position.

Pump Count Status

This area displays the current pump stroke count for each of the two pumps per drum. The accumulated value may be reset and the notification preset may also be configured.

Pump 'x' Count Reset

This button resets the accumulated pump count for the associated pump. This should be done after required maintenance has been performed.

Notify

This is the preset number of pump stroke counts. When this many strokes have been counted, the system will notify the user that it is time to perform required maintenance. Refer to the supply system documentation for maintenance procedures.

Actual

This is the number of pump strokes since the last time the count was reset.

Click Close to return to the View Menu.

Ghost Job Status

This screen indicates there are conditions to be met before a ghost job can be run. The text in green indicates a condition is met and red text indicates a problem area.  All conditions in the box below must be true (green) for a ghost job to proceed.

If  the above conditions have been met, a dialog box similar to the following will appear.  From here the PAINTworks user can request a Ghost job based on the settings they wish to run.  The left column displays the SIM CONVEYOR speed.  It is important that all these values are equal if you plan on using the SIM (Simulate) CONVEYOR option.

Click CANCEL to return to the Booth Status Display screen.

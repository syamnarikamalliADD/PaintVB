Cell Status Display - Robot Status

The purpose of this screen is to display the PAINTworks system information. This screen displays the booth status, job queue, process parameters, valve status and robot status. The operator, by utilizing several sub-screens, can view all of PW3 production status parameters. The operator is also allowed to edit the job queue and run ghost jobs from this screen.

Note: Changes to this screen can only be made by a user with sufficient access level.

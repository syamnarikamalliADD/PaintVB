Robot Life Cycle Display

This screen displays individual robot life cycle data.  To edit the fault or tolerance data select the current data and type the desired value in the space provided.  In the field that was changed double click and select Save from the menu that is displayed to save any changed data.  To reset the accumulated cycles for a valve or e-stat cable, double click in either the fault or tolerance field and select Reset from the menu that is displayed.

Click Booth to return to the Booth Status Display screen.

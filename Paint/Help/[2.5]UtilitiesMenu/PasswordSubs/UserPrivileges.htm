User Privileges

This screen is used to edit a user’s access to PAINTworks screens. Privileges that are enabled for the selected user are shown in Green. Disabled privileges are shown in Red.

All production data PW3 screens have password protection. The PW3 Password Protection allows an operator with a valid password (alphanumeric characters) to have access to PW3 screen functions. A PW3 supervisor or administrator assigns the  screen functions that operators can have access to.   The Change Password tab allows the user that is currently logged in to changes his/her password. Each user is given a list of functions he/she can perform as listed in the User Privileges tab.  The Group Manager tab provides a method to create sets of user privileges suited to various job functions. When new users are added, these privilege sets can then be applied to the user with one action rather than selecting privileges one at a time. The Setup tab is used to configure Auto Logout settings for all PAINTworks users.

Note: Changes to this screen can only be made by an operator with sufficient access level.

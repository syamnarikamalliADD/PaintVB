Setup

This screen is used to configure Auto Logout settings for all PAINTworks users. Auto Logout helps to protect PAINTworks from changes made by unauthorized personnel by automatically logging out the current user if PAINTworks is left unattended for the specified time interval. A warning dialog is shown prior to Auto Logout to allow the current user to maintain their login.

Note: Changes to this screen can only be made by an operator with sufficient access level.

Click the Area where you want help.

Screens/Screen Actions

This screen is used to add or remove PAINTworks screens from the security database, add or remove PAINTworks screen privileges from the security database, and assign privileges to PAINTworks screens. This screen is intended for use by the system integrator responsible for installing PAINTworks or adding/removing screens from an existing PAINTworks application and therefore is not required for daily operations. Changes to the data on the screen can adversely affect the operation of the PAINTworks system.

All production data PW3 screens have password protection. The PW3 Password Protection allows an operator with a valid password (alphanumeric characters) to have access to PW3 screen functions. A PW3 supervisor or administrator assigns the  screen functions that operators can have access to.   The Change Password tab allows the user that is currently logged in to changes his/her password. Each user is given a list of functions he/she can perform as listed in the User Privileges tab.  Privileges for each screen can be setup in the Screens/Screen Actions tab. The User Group Manager tab provides a method to create sets of user privileges suited to various job functions. When new users are added, these privilege sets can then be applied to the user with one action rather than selecting privileges one at a time.

Note: Changes to this screen can only be made by an operator with sufficient access level.

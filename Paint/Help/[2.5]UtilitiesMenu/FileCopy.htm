Utilities Menu

File Copy

This screen is used to backup and restore robot files. The tabs allow configuration of the backup.

  • Source - Select the source of the files. Select robot controllers to do a backup. Select the backup files to restore those files to the controller. If the controller is your source, you can also select the file device to read from. MDB: is the standard memory device with all the teach pendant and variable files. MD: has all the files from MDB: plus translations into text files. Additional choices are only for users familiar with the robot controllers. The drop-down list provides several default folders for backups and you may also choose to browse to any folder or zip file to select the source files.
  • Select Files - Select the files to copy. If "All Files" are selected on the source tab, this will not be used.
  • Destination - Select the destination for the files. Select robot controllers to restore files. Select a folder or zip file to perform a backup.
  • Do Copy - This tab displays a summary of the operation. Press the "Do Copy" buttont to start.

Delete Function

Use this button to delete files from any of the source locations.

Utilities Button

Several default choices are available from the utilities button. The drop down menu from the utilities button offer several default backup choices. These functions set all the settings requireed for the backup and select the "Do Copy" tab.

  • Backup All to Temp Backups - Select all the default backup files on the robots for the source. Select "Temporary Backups" as the destination.
  • Backup All to Temp Backups With Text Files - Select all the files on the robots for the source. Select "Temporary Backups" as the destination.
  • Backup All to Master Backups - Select all the default backup files on the robots for the source. Select "Master Backups" as the destination.
  • Backup All to Master Backups With Text Files - Select all the files on the robots for the source. Select "Master Backups" as the destination.
  • Get Easy Debug - Get the zipped easy debug files from the robot. Select "Temporary Backups" as the destination.
  • Backup Text Files - Select the text file translations of variable and teach pendant files on the robots for the source. Select "Temporary Backups" as the destination.
  • View/Compare - Open the compare window without selecting a file first.

Note: Changes to this screen can only be made by a user with sufficient access level.

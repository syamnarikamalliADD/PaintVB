Utilities Menu

Archive

This screen is used to backup PAINTworks databases, robot files, application, and DMON data. Several tabs provide backup functions for different components of PAINTworks. Each tab provides functions to backup it's files to a folder or zip archive.  Some tabs may not be present on all systems.

  • Managed Databases (SQL Server) - SQL Server manages these databases. The backup function on this tab creates a backup in the zone database folder (C:\Paint\Database\{Zone name}\Backup\) and copies the backup file to the archive location. The backup file can be restored using the SQL server. Databases may also be exported to CSV format which can be read in most spreadsheet programs and text editors.
  • Access Databases - Access databases are backed up by copying the file to the archive location. They can be read in Microsoft Access. They can be restored by copying the backup file to the zone database folder - C:\Paint\Database\{Zone name}\
  • XML Data - XML databases are backed up by copying the file to the archive location. They can be read in a text editor. They can be restored by copying the backup file to the zone database folder - C:\Paint\Database\{Zone name}\XML\
  • Image Backup Configuration - Image backup configuration files are backed up by copying the file to the archive location. They can be read in a text editor. They can be restored by copying the backup file to the zone database folder - C:\Paint\Database\{Zone name}\ECBR\
  • Notepad - Storage for the Notepad utility. These files can be restored by copying the backup file to the zone database folder - C:\Paint\Database\{Zone name}\Notepad\
  • Robots - Robot backups are created using the File Copy Utility screen, PAINTworks scheduled maintenance, or image backups initiated at the robot teach pendant. This tab allows each type of backup to be archived. Most backups can be restored directly from the archive using the File Copy Utility. Image Backups can be restored by copying them to the image backup folder (C:\Paint|Robot Image Backups\) before initiating an image restore at the teach pendant.
  • Application - This tab creates backups of the PAINTworks Application installation.
  • DMON Data - This tab Accesses DMON logger files.
  • DMON Data Archive - This tab Accesses DMON logger Archive files.

The utilities button also provides options to backup all the database tabs at once - Managed Databases, Access Databases, XML, Image backup Configuration, and Notepad.

Note: Changes to this screen can only be made by a user with sufficient access level.

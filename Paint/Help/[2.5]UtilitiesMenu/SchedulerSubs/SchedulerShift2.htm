Scheduler (Shift 2 Schedule)

This screen is used to setup shift 2 schedules. These shift schedules are used by the GUI to log alarms, downtime and production data. PW3 reports can be based on these shift start and end times. The Scheduler will log 3 shifts per day with up to five breaks in each shift. Shifts and break times can be enabled/disabled depending upon use. 

Note: Changes to this screen can only be made by an operator with sufficient access level.

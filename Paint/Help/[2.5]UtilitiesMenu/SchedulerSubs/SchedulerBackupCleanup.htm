Scheduler (Cleanup/Backup)

This screen is used to setup computer cleanup and file backup times. Cleanup operations include database compression, database backup and elimination of expired data from database tables. When the Robot Backup operation is performed, all process related files from each robot are copied to the hard drive. The File Backup operation backs up files selected on the File Backup Options tab to the selected device. This device could be a hard drive, network drive, or removable media such as a removable hard drive or USB thumb drive.

Note: Changes to this screen can only be made by an operator with sufficient access level.

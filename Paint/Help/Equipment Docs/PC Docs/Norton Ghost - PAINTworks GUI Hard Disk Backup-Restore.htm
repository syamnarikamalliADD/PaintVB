Overview

 

                The PAINTworks GUI computer supplied with the paint system is equipped with a CD-RW drive as well as additional hardware and software for backup and disaster recovery purposes.

 

                In addition to the  Primary (C:) hard drive, a Backup (D:)  hard drive is installed. The backup hard drive is the hardware equivalent of the primary drive and can be used to replace the primary drive in the case of a hard drive failure.

 

                The Symatec Norton Ghost software application and the Roxio Easy CD Creator software application are  installed in the primary C: drive. The Ghost application is used to create hard drive image files on either the backup hard drive or directly to CD media on supported CD-RW drives. Roxio Easy CD Creator is used to copy image files from the backup hard drive to CDs in the event that the CD-RW drive is not compatible with Norton Ghost .

 

                The following sections describe the steps required to create an image backup of the primary hard drive and restore the data from the backups to a hard drive. PAINTworks must be shut down and the operator must be able to log into the PAINTworks GUI as an Administrator in order to perform these functions. It is assumed in this document that the CD-RW drive is not compatible with Norton Ghost for direct write to CD.

 

Creating a Norton Ghost 2003 Boot Disk Set

 

                In addition to creating an image backup of the primary hard drive, a Ghost Boot floppy disk set must be created in order to restore an image to the hard drive in the event of a system failure. To create a boot floppy disk set, two blank 1.44Mb floppies will be required.

 

1.        Insert a Blank Floppy Disk in Drive A.

2.        From the Start Menu, select Programs/Norton Ghost /Norton Ghost or Right-Click on the Ghost tray icon and select Launch Norton Ghost.

3.        Select Ghost Utilities from the left side menu.

4.        Click on the Norton Ghost Boot Wizard Icon.

5.        Select CD/DVD Startup Disk with Ghost and click Next.

6.        Select Use PC-Dos when prompted to choose a DOS version and click Next.

7.        Select the Default Program Location "C:\Program Files\Symantec\Norton Ghost 2003\Ghost.exe" and click Next.

8.        Select Floppy Disk Drive = A:

9.        Number of Disk Sets to Create = 1

10.     Select Format Disk(s) first and Quick Format.

11.     Click Next.

12.     Click Next on the Screen that displays your selected options.

13.     Windows XP will now prompt you to format your floppy disk.  Click Start to Format the disk in Drive A.

14.     Click OK to continue.

15.     Click OK again when the format is complete.

16.     Close the format utility and the Ghost Boot Disk wizard will resume.

17.     Norton Ghost Boot Wizard will now copy the necessary files to the floppy disk and prompt for a second disk.

18.     When Ghost has finished writing the second disk, select FINISH to complete the Ghost Boot Disk Creation.

Backing Up Your PAINTworks GUI

 

                The following procedure details how to create an image file of the primary hard drive (C:) on the secondary hard drive (D:). The image file will be split into a number of 600Mb or smaller span files that can be copied to CD-R media. In this example the image files are created in the root directory of the backup (D:) drive but they can also be created in your own folder (example D:\GHOST\) on the backup drive. Image backups should be made whenever PAINTworks software changes are made or whenever new hardware or software is installed in the GUI.

 

1.        From the Start Menu, select Programs/Norton Ghost 2003/Norton Ghost  then click on the Backup Icon or Right-Click on the Ghost tray icon and select Backup your Computer.

2.        Click Next on the Backup Wizard window to continue and bring up the Backup Disk or Partition window.

3.        Select Disk 1 (C:) as the Source and File as the Destination, then click on Next to bring up the Create a New Backup Image window.

4.        Enter a filename and location on the D: backup drive for the image file in the Image File Name field (example D:\P500). Ghost will automatically add the .GHO file extension when the image file is created. Click Next to continue and bring up the Advanced Settings window.

5.        Click the Advanced Settings button.

6.        Click on the Compression tab and select High.

7.        Click on the Ghost Command Line tab and enter "-split=600" into the text box. This splits the image file into 600Mb files that can be written to CDR media.

8.        Click on the Apply button, then click on the OK button.

9.        Click the Next button on the Advanced Settings window. Ghost will inform you that your computer will be re-started in DOS mode to create the image backup. Click Next.

10.     Click Continue on the Disaster Recovery window

11.     Ghost will now re-start your computer and create an image of your PAINTworks disk. When this operation is complete, your computer will reboot back into Windows.

 

Copying Your Ghost Image Files to CDs.

1.        From the Windows XP Start Menu select Programs\Roxio Easy CD Creator 5\Project Selector.

2.        Select Make a Data CD and then Data CD Project from the Select a Project menu.

3.        Select the folder on the backup hard drive where the Ghost image files are stored in the Select Source Files dropbox (example D:\ or D:\Ghost).

4.        The main Ghost image file will have a .GHO extension (example P500.GHO). Subsequent spanned files will have a .GHS extension with a 3 digit spanned disk sequence number in the filename (example P500001.GHS would be the first spanned file, P500002.GHS would be the second and so on).

5.        Label a blank 700Mb CD-R disk PW3 Image 1 of n and insert it in the CD-RW drive .

6.        Select (highlight) the first ghost image file (example P500.GHO), then click the Add (down arrow) button. Next, click the Record button to copy this file to the CD. Remove the CD from the drive when the operation is complete.

7.        In the same fashion label additional CDs and copy each of the spanned (.GHS) files to them until all of the spanned image files have been copied to CDs.

8.        Store this backup file set in a safe place.

 

Restoring Your PAINTworks GUI

 

                The following procedure details how to restore a PAINTworks GUI primary hard drive from a disk image. The disk can be restored from CD ROMs or from image files stored on the backup hard drive. Note that data files such as Microsoft Access database files and robot backup files in the image backup may be out of date. PAINTworks is configured to make scheduled backups of  these files to the backup (D:) drive (or other user specified location). An Archive utility also is available from the PAINTworks Utilities menu that allows an operator to archive data files to a floppy disk. It is advisable to put a maintenace program in place that assures that these files are periodically backed up manually and stored in a safe location so they are available in the event of a system failure. PAINTworks data files from scheduled backups or archives should be used to replace out of date data files after a disk image is restored.

 

                In the event that a primary hard drive hardware failure made it necessary to perform this restoration, the primary hard drive must be replaced with a compatible hard drive with the equivalent or greater storage capacity.

 

                In the case where a replacement hard drive is not immediately available, the backup (D:) drive can be used by removing the data cable connector from both the C and D drives and then plugging the connector that was in the C drive into the backup drive. Warning: All data on the backup drive will be lost when the image is restored. If the only recent backups of PAINTworks data files reside on the backup drive, an effort should be made by a qualified computer technician to retrieve them prior to executing this procedure.

 

                This procedure can only be used on the original or identical GUI hardware. If the GUI computer must be replaced with another model due to hardware failure, contact FANUC Robotics America, Inc. for assistance.

 

1.        With the GUI computer powered off, insert disk 1 of  your Symantec Norton Ghost Boot Disk Set in drive A. Switch computer power on.

2.        The computer will load DOS from the floppy disk. It will also attempt to load drivers for a number of CDROM drive types. Most of these drivers will fail to load because these drive types are not installed. Please ignore any related error messages. You will be prompted to insert the second floppy disk of the Boot Disk Set when it is required.

3.        If you are restoring from CDs, insert the first CD of your spanned backup into the CD-RW drive.

4.        Wait for the computer to load the Norton Ghost utility, then click OK on the About Norton Ghost screen.

5.        From the displayed menu, select LOCAL, select DISK and then select FROM IMAGE.

6.        Using the Look In drop down box, select the CD-ROM drive or other location where image files are located.

7.        Select the image file with a .GHO extension as the image source.

8.        Select Drive 1 as your destination and click OK.

9.        Click OK to select the default Destination Drive Details.

10.     Norton Ghost will prompt you if you'd like to proceed with disk load.  Click yes to continue.

11.     Follow Norton Ghost's prompts for subsequent CDs until the restore is complete. When restoring from spanned image files in the same folder on a hard drive, Click OK when Ghost prompts for the next spanned image file.

12.     Remove any floppy disks from the disk drive and any CDs from the CD-ROM drive and restart the computer.

13.     From the Windows desktop, using Explorer, replace out of date PAINTworks database files in C:\Paint\Database with files from a recent archive.

14.     Re-start PAINTworks.

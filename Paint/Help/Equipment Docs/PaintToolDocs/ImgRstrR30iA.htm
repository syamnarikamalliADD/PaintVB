Restoring an Image Backup

This procedure will restore an image backup onto a controller. It will completely erase what is currently on the controller. Be sure to do a full backup with PAINTworks first.

1) With the power off, press and hold the F1 and F5 keys on the teach pendant. While holding these keys, press the ON button. Wait about ten seconds, or until you see the BMON Menu (below) on the teach pendant before releasing the buttons.


***BOOT MONITOR ***
Base Version V7.50P/10 [Release 2]
Initializing file devices ... done.
1. Configuration menu
2. All software installation
3. INIT start
4. Controller backup/restore
5. Hardware diagnosis
6. Maintenance
7. All software installation(Ethernet)
8. All software installation(USB)

Select : _

2) Select: 4. Controller backup/restore to see the backup/restore menu:


***BOOT MONITOR ***
Base Version V7.50P/10 [Release 2]
**** BACKUP/RESTORE MENU  *****
0.   Return to Main Menu
1.   Emergency Backup
2.   Backup Controller As Images
3.   Restore Controller Images
4.   Bootstrap To Cfg Menu

Select : _

3) Select: 3. Restore Controller Images :


*** BOOT MONITOR***
**** Device selection menu  *****
1.  Memory Card(MC:)      
2.  Ethernet(TFTP:)
Select : _

To restore from PAINTworks computer over ethernet, skip to step 7.

MEMORY CARD MEMORY CARD RESTORE

4) To restore from memory card select: 1. Memory Card(MC:) :


*** BOOT MONITOR***

**** RESTORE Controller Images  *****
     Current module size:
      FROM: 64 mb  SRAM: 3 mb

     CAUTION:   You SHOULD have image files
      From the same size of FROM/SRAM.
      If you don't, this operation will cause
      Fatal damage to the controller.

      Are you ready?[Y=1/N=else]:

5) Place a memory card with the controller images into the slot on the controller. Press 1, ENTER. The image files will be written to the controller. You should see the following messages:

Checking MC:\FROM00.IMG.Please wait..
*************************************
  CAUTION: NEVER TURN OFF THE POWER
  SUPPLY WHILE CLEARING FROM !!!
*************************************
Clearing FROM ...Done
Clearing SRAM (3M) ... done
Reading MC:\FROM00.IMG ... Done
...
Reading MC:\SRAM02.IMG ... Done
6) Press ENTER to continue.

All the teach pendant programs and variable settings are restored from when the backup was made. If it is a recent backup from the same robot, then you may not need to restore from the PAINTworks file copy utility.

If the image backup is from the same robot, but older; You should restore files from the PAINTworks.

If the image is from a different robot, then you will need to run the setup program before you restore the variables.

If the image is from a different robot, then you will need to run the setup program before you restore the variables.

Making an Image Backup

This procedure will save an image backup to a memory card or the PAINTworks computer. It will use 35MB. Once the backup is made, It can be used to load software on any robot with the same configuration, but it is easiest to use an image from the robot you are loading.

1) With the power off, press and hold the USER1 and USER2 buttons on the operator panel of the RJ3 controller or press and hold the F1 and F5 keys on the teach pendant. While holding these keys, press the ON button. Wait about ten seconds, or until you see the BMON Menu (below) on the teach pendant before releasing the buttons.


***BOOT MONITOR for R-J3(iB/ibMate) CONTROLLER***
Base System Version V6.4303 (FRNA)
Initializing file devices ... done.
1. Configuration menu
2. All software installation
3. INIT start
4. Controller backup/restore
5. Hardware diagnosis
6. Maintenance
7. All software installation(Ethernet)

Select : _

2) Select: 4. Controller backup/restore to see the backup/restore menu:


***BOOT MONITOR for R-J3(iB/ibMate) CONTROLLER***
**** BACKUP/RESTORE MENU  *****
0.   Return to Main Menu
1.   Emergency Backup
2.   Backup Controller As Images
3.   Restore Controller Images
4.   Restore Full Ctlr Backup (From MC)
5.   Bootstrap To Cfg Menu

Select : _

3) Select: 2. Backup Controller As Images :


***BOOT MONITOR for R-J3(iB/ibMate) CONTROLLER***
**** Device selection menu  *****
1. Memory Card(MC:)
2. Ethernet(TFTP:)

Select : _

For backup to PAINTworks computer over ethernet, skip to step 6.

MEMORY CARD BACKUP

4:) For backup to memory card select: 1. Memory Card(MC:) to start the backup:


***BOOT MONITOR for R-J3(iB/ibMate) CONTROLLER***
**** BACKUP Controller as Images  *****
     Module size to backup:
     FROM: 32 mb  SRAM: 3 mb

     Please Insert MC: which has at least
     1Mb free space.
     Are you ready?[Y=1/N=else]:

5) Place the memory card in the slot on the controller. Press 1, ENTER to continue.
The teach pendant will display status messages while it writes to the card
Writing MC:\FROM00.IMG (1/35)
If the card does not have enough free space, it will ask for another.

When it is finished it will say it is done. Press ENTER to continue. Turn the controller off then on to return to production.

ETHERNET BACKUP

6:) To start backup over ethernet to PAINTworks computer select: 2. Ethernet(TFTP:) :


***BOOT MONITOR for R-J3(iB/ibMate) CONTROLLER***
**** BACKUP Controller as Images  *****
     Module size to backup:
     FROM: 32 mb  SRAM: 3 mb

   Disk spave required on remote host is
     35Mb free space.
     Are you ready?[Y=1/N=else]:

7) Press 1, ENTER to continue.
The teach pendant will display status messages while it writes to the computer:
Writing MC:\FROM00.IMG (1/35)

When it is finished it will say it is done. Press ENTER to continue. Turn the controller off then on to return to production.

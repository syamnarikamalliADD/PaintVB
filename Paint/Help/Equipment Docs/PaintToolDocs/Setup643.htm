IO Setup and Variable Restore

This procedure will setup the communication with the PLC and GUI, it can be used to restore variables after an image was loaded from another robot with the same options.
If you don't need to change the robot number and only want to restore the variables or run the I/O setup, start at step 5

Select the Robot and Zone

1) With the power off, press and hold the USER2 button on the operator panel of the RJ3 controller or press and hold the PREV and NEXT keys on the teach pendant. While holding these keys, press the ON button. Wait about ten seconds, or until you see the Configuration Menu (below) on the teach pendant before releasing the buttons.


----- CONFIGURATION MENU -----

1. Hot start
2. Cold start
3. CNTL start
4. Maintenance

Select >

2) Select: 3. Controlled Start It will bring you to the PaintTool setup screen:

   I/O Configuration:        Enhanced
7  Cell I/O Hardware:        Ethernet I/P
8  Process I/O Hardware:     Model A (R1-10)
9  No. of System Colors:     35
10 No. of Color Valves:      31



Set the controller number:
Half the 2nd robot number; Robots 1 and 2 = Controller 1, Robots 7 and 8 = Controller 4.
Set the Zone number to Clearcoat.
Press F2(Setup) to run the setup. It will take a few minutes.

4) When it finishes it will display "Press FCTN then START (COLD) when done." Press FCTN, then ENTER to cold start. It will take a few minutes to finish.

Variable Restore

5) With the power off, press and hold the USER2 button on the operator panel of the RJ3iB controller or press and hold the PREV and NEXT keys on the teach pendant. While holding these keys, press the ON button. Wait about ten seconds, or until you see the Configuration Menu (below) on the teach pendant before releasing the buttons.


----- CONFIGURATION MENU -----

1. Hot start
2. Cold start
3. CNTL start
4. Maintenance

Select >

6) Select: 3. Controlled Start It will bring you to the PaintTool setup screen:

PaintTool Setup       CONTROLLED START

9  No. of System Colors:     35
10 No. of Color Valves:      31

11) You may now restore variables and teach pendant programs (*.SV, *.VR, and *.TP) from the PAINTworks file copy utility. Wait for PAINTworks to finish.

12) Press FCTN, then ENTER to cold start. It will take a few minutes to finish.
It will end up at the utilities menu.

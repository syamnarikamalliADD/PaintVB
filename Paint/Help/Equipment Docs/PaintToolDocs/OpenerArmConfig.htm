Opener Arm Extention Configuration

This procedure will setup the robot software when the vertical extension is changed on the P-20iA robot

1) With the power off, press and hold the PREV and NEXT keys on the teach pendant. While holding these keys, press the ON button. Wait about ten seconds, or until you see the Configuration Menu (below) on the teach pendant before releasing the buttons.


----- CONFIGURATION MENU -----

1. Hot start
2. Cold start
3. CNTL start
4. Maintenance

Select >

2) Select: 3. Controlled Start It will bring you to the PaintTool setup screen:

3) Press MENU, 9 for the Robot Maintenance screen.

ROBOT MAINTENANCE


  Setup Robot System Variables

   Group   Robot Library/Option Ext Axes
     1     P-700iA/15P             1
     2     P-20iA iC               1
     3     P-20iA iC               1
     0      Process Axes Control   2

4) Use the arrow keys to scroll to the motion group that needs to change and press F4 MANUAL Select the default (current) answers for all questions until it asks:

Is the J3 Extension installed?
Enter 1 for no extension, 2 for an extension.

5) Repeat step 4 for the opener if needed.

6) Press FCTN, then ENTER to cold start. It will take a few minutes to finish.

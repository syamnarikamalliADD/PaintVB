Making an Image Backup

This procedure will save an image backup to a memory card or the PAINTworks computer. It will use 67MB. Once the backup is made, It can be used to load software on any robot with the same configuration, but it is easiest to use an image from the robot you are loading.
Image backup can be started from the teach pendant file menu, or from the boot monitor menu.

A. Start An Image Backup from the File Menu

1)Enable the teach pendant.  If you are making a memory card backup, put the memory card in it's slot in the R-30iA CPU.

2)Press MENU, then select 7 FILE. to get to the file menu.

FILE
   MC:\*.*
     1 *      *   (All Files)
...
[TYPE] [DIR] LOAD [BACKUP] [UTIL]
3)Press F4 [BACKUP] and select Image Backup.

Destination Device
  1. Memory Card(MC:)
  2. Ethernet(TFTP:)
4)Select
    1 Memory Card(MC:) to back up to a memory card in the controller CPU.
    2 Ethernet(TFTP:) for a backup directly to the PAINTworks computer over the network.

Remove existing IMG files ?
                       YES    NO
5)If you see this prompt, Press F4 (YES) to confirm

Cycle Power?
                        OK    Cancel
6)Press F4 (OK) to confirm.  The controller will shut down to start the backup.

****Auto image backup start ****

Checking *.IMG files on MC:\

 **** BACKUP Controller Images ****
Writing MC:\FROM...
The Controller will reboot when it finished making the backup.

FILE
   MC:\*.*
     1 *      *   (All Files)
...
Image backup completed successfully.
                  OK
7)Press F4 OK to continue using the robot.

C. Memory Card Backup

1) For backup to memory card select: 1. Memory Card(MC:) to start the backup:


***BOOT MONITOR***
**** BACKUP Controller as Images  *****
     Module size to backup:
     FROM: 64 mb  SRAM: 3 mb

     Please Insert MC: which has at least
     1Mb free space.
     Are you ready?[Y=1/N=else]:

2) Place the memory card in the slot on the controller. Press 1, ENTER to continue.
The teach pendant will display status messages while it writes to the card
Writing MC:\FROM00.IMG (1/35)
If the card does not have enough free space, it will ask for another.

When it is finished it will say it is done. Press ENTER to continue. Turn the controller off then on to return to production.

D. Ethernet Backup

1) To start backup over ethernet to PAINTworks computer select: 2. Ethernet(TFTP:) :


***BOOT MONITOR***
**** BACKUP Controller as Images  *****

   Disk spave required on remote host is
     67Mb free space.
     Are you ready?[Y=1/N=else]:

2) Press 1, ENTER to continue.
The teach pendant will display status messages while it writes to the computer:
Writing MC:\FROM00.IMG (1/35)

When it is finished it will say it is done. Press ENTER to continue. Turn the controller off then on to return to production.

Overview

 

                The FANUC Robotics PAINTworks system includes an Allen-Bradley Logix 5563 PLC. This document describes the steps required to create a backup of the PLC program on the GUI hard drive and to restore the PLC program to the PLC from  the GUI hard drive using Rockwell RSLogix 5000 software. A current backup of the PLC program is stored in the C:\Paint\PLC folder on the PAINTworks GUI hard drive.

 

Backing Up Your Logix 5563 PLC Program

 

                The following procedure details how to create a backup of the Logix 5563 PLC program on the PAINTworks GUI hard drive using Rockwell RSLogix 5000 software. PLC backups are not automatically created whenever logic changes are made. This procedure should be used if changes have been made to the PLC program or if an attempt is made to go Online and Logix 5000 presents a dialog that states "The open project has offline changes that aren't in the controller".

 

Note: Backup files should also be copied to removable media and stored in a safe location so they are available in the event of a system failure. RSLogix 5000 software must be shut down before the <site-specific>.ACD file can be copied.

 

1.        From the PAINTworks Maintenance Menu, click on RSLogix 5000.

2.        The PLC program folder should load automatically. If not, from the File menu, select Open. Then, select C:\Paint\PLC\<Site-Specific>.ACD from the Open dialog box.

3.        From the RSLogix Communication menu, choose Go Online.

4.        If a dialog box appears that states "The open project has offline changes that aren't in the controller" and you wish to back up the program that is currently on the PLC, click on the Upload button. Otherwise, choose Save from the RSLogix 5000 File menu or click on the  floppy disk icon on the RSLogix 5000 toolbar. You will be asked if you wish to upload tag (data table) values. It is a good idea to click on Yes for this option.

5.        The PLC will store the backup (.ACD) file in the C:\Paint\PLC folder on the GUI computer.

 

Restoring Your Logix 5563 PLC Program

 

                The following procedure details how to restore the Logix 5563 PLC program from a backup on the PAINTworks hard disk using RSLogix 5000 software. This procedure will differ depending on whether the PLC has no program loaded or  the PAINTworks PLC program is already loaded on the PLC.

 

If no program is loaded

1.        Place the mode key switch on the Logix processor in the (right) Program Mode position.

2.        Connect a Nine-Pin serial programming cable from the GUI COM1 Serial Port to the serial port on the Logix 5563 processor.

3.        From the PAINTworks Maintenance Menu, click on Logix 5000.

4.        The PLC program (<site-specific.ACD>file should load automatically. If not, from the File menu, select Open. Then, select C:\Paint\PLC\<Site-Specific>.ACD from the Open dialog box.

5.        From the Logix 5000 Communications menu, choose Select Recent Path. Choose the <site-specific controller name> AB_DF1-1 connection from the dialog box, then click  the Go Online button.

6.        A dialog box will be presented that states "there is no project loaded in the controller".  Click on the Download button to load the program. Another dialog box to confirm the download will be shown. Click on the Download button on this dialog to resume the download. The download will take about 5 minutes to complete.

7.        When the download to PLC is complete, the PLC can be placed in Run mode by placing the mode key switch on the Logix processor in the (left) Run Mode position and then returning the switch to the center Remote Mode position.

8.        From the Logix 5000 Communications menu, choose  Go Offline.

9.        Unplug the serial cable from both the GUI computer COM1 port and the Logix processor serial port.

10.     From the Logix 5000 Communications menu, choose Select Recent Path. Choose the <site-specific controller name> TCP-1\ <ethernet IP Address\Backplane\0 connection from the dialog box, then click  the Go Online button.

 

If a program is loaded

  1. Place the mode key switch on the Logix processor in the (right) Program Mode position.
  2. From the PAINTworks Maintenance Menu, click on Logix 5000.
  3. The PLC program (<site-specific.ACD>file should load automatically. If not, from the File menu, select Open. Then, select C:\Paint\PLC\<Site-Specific>.ACD from the Open dialog box.
  4. From the Logix 5000 Communications menu, choose Select Recent Path. Choose the <site-specific controller name> TCP-1\ <ethernet IP Address\Backplane\0 connection from the dialog box, then click  the Download button.
  5. A dialog box to confirm the download will be shown. Click on the Download button on this dialog to start the download. The download will take less than 1 minute to complete.
  6. When the download to PLC is complete, the PLC can be placed in Run mode by placing the mode key switch on the Logix processor in the (left) Run Mode position and then returning the switch to the center Remote Mode position.

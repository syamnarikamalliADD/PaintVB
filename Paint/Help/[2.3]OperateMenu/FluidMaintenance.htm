Operate Menu

Fluid Maintenance

The purpose of this screen is to allow the operator to perform manual flow tests, run color change cycles and to control individual color change valve states.

To request a manual fluid function, the operator must first select the color and robot(s). Flow test and valve commands can be sent to multiple robots simultaneously but the current robot valve state is displayed one robot at a time. The operator can enable Individual valves by clicking on the graphical representation of the color valve stack. When valves are selected to be on and the feedback from the robot controller goes on, the GUI icon will be highlighted. When the valve is disabled (toggle state) and the feedback from the robot controller goes off, the GUI icon returns to it’s normal color.

The graphical valve area can display other flow information. Right click to bring up a menu that allows a choice of displays.

Flow tests are started by first entering the flow test parameters into the Flow Test text boxes. Data for Paint Flow (cc), Turbine Speed (KRPM), Shaping Air Pressure (SLPM), Estats (step) and Flow Test Time Duration (seconds) can be entered. The operator starts the flow test by pressing the Start button. The test is aborted by pressing the Cancel button. When a flow test is active, the flow test status bar shows the percentage of the flow test complete. During a flow test, instantaneous and total feedback values are displayed.

Some parameters can also be sent as output counts instead of engineering units. Right click on the units label to select Engineering units or counts. This applies to most analog or group output parameters. It is not available for Servo-driven fluid control or Fanuc bell speed control.

A manual color change cycle is started by selecting and individual color change cycle and pressing the Run [Color Change] Button. Any valves energized during the color change cycle are displayed by the graphical representation of the color changer valve stack.

Autocal can be run for any parameter that supports it(Accuflow, Accustat, DQ Shaping Air). The graphical valve area will change to display autoacal status.

Note: Changes to this screen can only be made by a user with sufficient access level.

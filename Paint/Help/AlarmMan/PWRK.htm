PWRK-001 MESSAGE Alarm Manager Started

Cause:
PAINTworks Alarm Manager has been re-started.

Remedy:
Status message only.

PWRK-002 MESSAGE Connection Established Alarm 002

Cause:
An Ethernet connection to this robot controller has been establised.

Remedy:
Status message only.

PWRK-003 WARNING Connection Lost

Cause:
The Ethernet connection to this robot controller has been lost. This may be due to powering down the robot controller, a disconnected Ethernet cable, or an Ethernet communication hardware failure.

Remedy:
Verify that the robot controller power is ON. Verify the integrity of Ethernet cable and connections. Verify the operation of the robot controller Ethernet communication hardware.

PWRK-004 WARNING Connection Unavailable

Cause:
Cannot establish an Ethernet connection to the robot controller. This may be due to powering down the robot controller, a disconnected Ethernet cable, or an Ethernet communication hardware failure.

Remedy:
Verify that the robot controller power is ON. Verify the integrity of Ethernet cable and connections. Verify the operation of the robot controller Ethernet communication hardware.

PWRK-005 WARNING In Controlled Start

Cause:
The robot controller is currently running in Controlled Start mode. Alarm status cannot be retrieved in this mode.

Remedy:
Status message only.

PWRK-006 WARNING Network Connection Lost

Cause:
The PAINTworks GUI Ethernet connection to the network has been lost. This may be due to a disconnected Ethernet cable or network interface card failure.

Remedy:
Verify the integrity of Ethernet cable and connections. Verify the operation of the GUI computer network interface card.

PWRK-007 WARNING Heartbeat Lost

Cause:
The robot controller has stopped communicating with the GUI computer.

Remedy:
This is typically a temporary condition that occurs when going from Controlled Start to Cold Start.

Change Log Summary

This screen displays information regarding modifications that have been made. The particular type of changes shown depend on which screen the user was on when the Change Summary button was pressed.

Click the area where you want help.

Description of Screen Controls

The following table provides a functional description of the various screen objects:

ITEM

DESCRIPTION

Close

Click on this button to close this screen and return to the previous screen.
Save Click on this button to save the report to a file.

Print

Click on this button to print the report.

Last 24 Hours

Click on this button to generate a report for the last 24 hours.

Last 7 Days

Click on this button to generate a report for the last 7 days.

All Changes

Click on this button to generate a report of all changes.

Zone

The zone number that the particular change applies to.

Area

This is the particular screen that the change was made on.

User

This is the user that was logged in when the change was made.

Specific

This is a description of the associated change.

Zone

This is the zone that was changed.

Device

The robot, controller, or zone that was changed

Parameter

Parameter in the change - color, style,....

Time

This is the time and date of the change.

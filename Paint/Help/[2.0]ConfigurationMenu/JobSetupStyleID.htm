Configuration Menu

Job Setup - Styles

The purpose of this screen is to configure job setup information. Tabs are used to select each job setup parameter: Style, Option, Degrade, Repair, Intrusion

Styles

The plant styles are assigned to Fanuc styles. Each style is assigned a description. Both the plant style and Fanuc style fields can be assigned the same number. However, each Plant Style field must be unique. Enter a 0 for any unused styles. The FANUC style is the style number used when the JOB is created. Each robot can be enabled or disabled for each style. Under the Robots Required field, a checked robot means that robot is required to paint the selected style while an unchecked robot is not. With the proper access level, a user can restore these styles from either the database or the PLC in the current zone.

The Style information is saved to the PLC. The PLC uses this data to verify that the Style is valid and to initialize the required robot controllers with the correct style number.

Note: Changes to this screen can only be made by a user with sufficient access level.

Configuration Menu

Job Setup - Options

The purpose of this screen is to configure job setup information. Tabs are used to select each job setup parameter: Style, Option, Degrade, Repair, Intrusion

Options

The plant options are assigned to Fanuc options. Each option is assigned a description. Both the plant option and Fanuc option fields can be assigned the same number. However, each Plant Option field must be unique. The Fanuc option is the option number used when the JOB is created. Each robot can be enabled or disabled for each option. Under the Robots Required field, a checked robot means that robot is required to paint the selected option while an unchecked robot is not. With the proper access level, a user can restore these styles from either the database or the PLC in the current zone.

The Option information is saved to the PLC. The PLC uses this data to verify that the Option is valid and to initialize the required robot controllers with the correct option number.

Note: Changes to this screen can only be made by a user with sufficient access level.

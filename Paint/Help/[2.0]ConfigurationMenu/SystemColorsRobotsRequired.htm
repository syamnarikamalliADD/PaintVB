Configuration Menu

System Colors - Robots Required

The purpose of this screen is to manage the color configuration. Tabs are used to select each sets of color information: Valves, Robots Required, and 2K configuration.

Robots Required

The purpose of this screen is to enable or disable each robot arm for each system color. Under the Robots Required field, a checked robot indicates that robot is required to paint the selected colour while an unchecked robot is not. With the proper access level, a user can restore system colors from either the database, the PLC or one of the robots in the current zone.

This Fanuc system color, description and valve assignment are saved to the robot controller. The robot uses this information to store and process system color and valve instructions. The Fanuc system color and valve assignment is also saved to the PLC. The PLC uses this data to verify that the job queue color is valid and to initiate the motion controllers with the correct color number.

Note: Changes to this screen can only be made by a user with sufficient access level.

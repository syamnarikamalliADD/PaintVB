Repairs

The purpose of this screen is to configure repair panel information. Each repair panel is  assigned a description. In the Robots Required field, a checked robot means that robot is required to paint the selected repair panel while an unchecked robot is not. With the proper access level, a user can restore these values from either the database or the PLC in the current zone.

The Repair Panel Robots Required information is saved to the PLC. The PLC uses this data to initialize the required motion controllers with the correct Repair Panel information.

Note: Changes to this screen can only be made by a user with sufficient access level.
